# v1.1.0

Version 1.1.0 focuses on resource management, project consistency, and internal reorganization to prepare for the stable multiplatform release.

## Key Changes

### Resource Management (Loan Pattern)
Introduced the `Disposable` and `DisposableAsync` interfaces to support explicit resource lifecycle management and non-blocking cleanup across all platforms.

### Project Consistency
Added a comprehensive `.editorconfig` to enforce unified coding styles, indentation, and formatting across the entire codebase, ensuring high code quality.

### Package Reorganization
Improved the internal project structure by regrouping modules into more specific categories:
- **DSL Pattern**: `com.davils.kore.pattern.creational.dsl`
- **Event Pattern**: `com.davils.kore.pattern.reactive.event`
- **System Environment**: `com.davils.kore.system.environment`

## Stability Updates
- **Java Compatibility**: Downgraded Java and JDK version from 25 to 17 in the build configuration to ensure better stability and compatibility with current CI/CD tooling.
- **Dependency Updates**: Bumped various internal and external dependencies including KSP and Gradle Wrapper.
